## Skills

- [`medical-app-ui`](skills/medical-app-ui/SKILL.md): skill para construir,
  redesenhar e revisar interfaces de servicos medicos e apps de saude.
  Compatível com Codex e Claude Code.

### Claude Code

Instale a pasta da skill em `~/.claude/skills/medical-app-ui`:

```bash
mkdir -p ~/.claude/skills
cp -r skills/medical-app-ui ~/.claude/skills/medical-app-ui
```

O arquivo portátil é `skills/medical-app-ui/SKILL.md`. A pasta
`skills/medical-app-ui/agents` mantém metadados extras para Codex.
